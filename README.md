## 📂 Featured Engineering Case Studies

### 1. 2D to 3D Medical X-ray Reconstruction Pipeline
* **Problem:** Medical 2D images lack depth info, making spatial anatomy assessment hard without heavy CT/MRI scans.
* **Process:** Deployed monocular depth estimation via **MiDaS**. Engineered custom processing filters to handle biological density values, mapped output depth matrices into Cartesian coordinates, and integrated an interactive Streamlit viewport allowing surgeons to zoom and measure structures in simulated 3D space.
* **Results:** Boosted reconstruction accuracy by **27%** over baselines with zero-latency mesh rendering speeds of **<1.2s**.

### 2. Intelligent Candidate Screening & NLP Ranker
* **Problem:** Recruiters spend excessive hours manually filtering hundreds of PDF resume applications.
* **Process:** Designed text preprocessing routines in SpaCy. Extracted features using TF-IDF and SentenceTransformers (BERT), and trained a classification model to score context relevance against target job metrics.
* **Results:** Created an objective decision scoreboard, accelerating initial candidate pre-screening velocities by **60%**.

### 3. Bank Forms Document Intelligence (OCR & Automated Routing)
* **Problem:** Paper-based banking slips (checks, deposits) cause high operational data input errors.
* **Process:** Built OpenCV filters (Deskewing, binarization, noise thresholding) to improve scanned text quality. Ran multi-threaded Tesseract OCR engines for high-volume extraction, and utilized supervised text classifiers to categorize documents.
* **Results:** Achieved a **92% OCR field accuracy** and **95% classification precision**, reducing manual processing loops by **50-60%**.

## 🚀 Experience the Portfolio Locally

The codebase compiles into a high-end React SPA utilizing Vite, Tailwind CSS, and Framer Motion. It features an interactive, spring-backed precision trailing custom pointer, layout borders that span widescreen bounds, and an **animated SVG Resume Bot** that is preloaded with real resume data.

### Running the App:
1. Ensure Node.js is installed.
2. Clone the repository and navigate to the directory:
   ```bash
   cd portfolio
   ```
3. Install standard packages:
   ```bash
   npm install
   ```
4. Run the Vite local development server:
   ```bash
   npm run dev
   ```
5. Open `http://localhost:5173` in your browser.
